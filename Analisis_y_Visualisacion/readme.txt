Para poder realizar este práctico es necesario tener instalado anaconda.

Existe una herramienta que funciona en Linux y windows que se llama Anaconda Navigator,
pueden descargarla en el siguiente enlace.

https://anaconda.org/anaconda/anaconda-navigator

Esta herramienta les permite crear entornos virtuales de python e instalar las librerias
necesarias para cada entorno. Además contiene herramientas como Jupyter, que es donde
corren las notebook sobre las que se implementarán los distintos prácticos.

Las librerias necesarias para esta primera practica son las que veran en el teórico:
 - pandas
 - seaborn
 - scipy
 - numpy

Estas librerías se podrán instalar, en nuestro entorno virtual, usando Anaconda Navigator.
